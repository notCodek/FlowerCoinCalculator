I don't know why you're here but well... This is the repo for the Flower Coin Calculator!

This is specifically to be used for Mario Wonder 100% speedruns, using the main route.
(Changes for the Switch 2 Edition will be added at some point)

Put in your Standee and Coin counts after the 4th shop, and then adjust your Future Coins with the slider to see how many coin galores you need for every Standee. Pretty standard stuff.
